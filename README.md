## General info
This project is simple calculator with minimalistic beautiful UI and dark-mode

## Technologies
Project is created with
* React Native version: 0.68.2
* Expo version: ~45.0.0

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start 
    OR
$ expo start
```
